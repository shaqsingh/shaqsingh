The Varsity

Responsive UI

In approaching the design, I wanted to create something that would be timeless, minimal, and responsive. I wanted to make sure that the site was flexible for future use and so used a 300 pixel wide sidebar throughout the site for extra content like ads, subsection lists, related articles, and anything else that should stand apart from the page's main content. I increased the font size throughout for easier reading and got rid of elements that weren't user friendly such as the carousel.

This site won second place in the Best campus website category of the 2015 Canadian Community Newspaper Awards.

The Varsity Magazine

Microsite UI

In approaching the design, I wanted to create something that would be flexible and transparent so that it could take on the design of the ever-changing theme. Each magazine has a different colour palette and unique design elements and I wanted the site to reflect that. The grid below the top 5 blocks is completely flexible, within the constraint that each row has 1,2, or 3 blocks. Should an article not have an associated visual, the block thumbnail is simply set to one from the respective magazine's colour palette. Articles can be flipped through via the keyboard or the arrows alongside each article.
